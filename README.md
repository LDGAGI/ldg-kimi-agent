# ldg-kimi-agent

个人为 Kimi Code CLI 编写的 `AGENTS.md` 配置文件。

## 说明

本仓库中的 `AGENTS.md` 是我根据自己的使用习惯和需求编写的 Kimi Code 代理配置，用于规范 Kimi Code 在终端环境中的行为、编码风格和工作流程。

## 来源与致谢

本配置文件参考并改写了 **CLAUDE-FABLE-5** 的系统提示。

- 原始来源：[elder-plinius/CL4R1T4S — ANTHROPIC/CLAUDE-FABLE-5.md](https://github.com/elder-plinius/CL4R1T4S/blob/main/ANTHROPIC/CLAUDE-FABLE-5.md)
- 项目主页：[https://github.com/elder-plinius/CL4R1T4S](https://github.com/elder-plinius/CL4R1T4S)

在此感谢 [elder-plinius](https://github.com/elder-plinius) 分享的相关资料。

## 使用方式

将 `AGENTS.md` 放置到项目根目录或 `~/.kimi/` 目录下，Kimi Code CLI 会自动读取并应用其中的规则。

- `AGENTS.md` — Kimi Code 代理配置文件
